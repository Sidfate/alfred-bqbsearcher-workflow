### 目录

```
.
├── FileToClipboard     // 拷贝到剪贴板的执行程序
├── README.md           // 项目说明
├── bin
│   └── build           // workflow 构建的执行程序
├── bqbsearcher.php     // 主功能文件
├── data.json           // 缓存的表情包数据库
├── icon.png            // workflow 的 icon
├── info.plist          // workflow 的配置信息
└── workflows.php       // workflow 开发插件类
```
